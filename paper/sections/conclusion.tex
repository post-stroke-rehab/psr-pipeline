\section{Conclusion}

This paper documents \repo\ as a reproducible workflow for sEMG-based finger-intent prediction in a post-stroke rehabilitation setting. The key idea is not a new standalone network, but a repository structure that makes preprocessing, feature extraction, tensor adaptation, model comparison, and metric reporting explicit and reusable.

The strongest current evidence favors the LSTM for exact-match accuracy, finger accuracy, and macro AUROC, while the GNN offers the best macro F1 and AUPRC. That trade-off makes the repository useful for controlled model comparison rather than single-number leaderboard claims. A current limitation is evaluation scope: the benchmark is still centered on one dataset and lacks stored ablations for several design choices. Extending the same workflow to broader baselines, richer transfer settings, and more clinically realistic validation is the clearest next step.
