Reliable finger-intent decoding from surface electromyography (sEMG) is attractive for post-stroke rehabilitation, but it is difficult to study reproducibly when preprocessing choices, tensor layouts, and model evaluation code are scattered across separate scripts. This paper documents \repo, a repository-centered workflow that packages a three-stage sEMG preprocessing stack, a twelve-feature window representation, shared model adapters, and benchmark utilities for multi-label finger-intent prediction on \physiomio~\citep{psrrepo2026,physiomio2026}. The current pipeline filters raw EMG with a 20--450 Hz zero-phase Butterworth stage, denoises it with wavelet soft-thresholding, segments 200 ms windows with 50\% overlap, and converts each window into a channel-by-feature tensor that can be consumed by LSTM, CNN, and graph neural network predictors. On the repository's held-out test artifacts, the LSTM achieves the strongest exact-match accuracy, finger accuracy, and macro AUROC (0.545, 0.784, and 0.858), while the GNN achieves the strongest macro F1 and AUPRC (0.706 and 0.776). These results position \repo\ as a useful systems benchmark rather than a single-model state-of-the-art claim, because the available evidence still covers one dataset and does not yet include full ablation or cross-dataset generalization studies. Even with that limitation, the released pipeline, metrics, and paper-support artifacts establish a concrete base for future rehabilitation-oriented sEMG experiments.
