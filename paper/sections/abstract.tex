Reliable finger-intent decoding from surface electromyography (sEMG) is attractive for post-stroke rehabilitation, but it is difficult to study reproducibly when preprocessing choices, tensor layouts, and model evaluation code are scattered across separate scripts. This paper documents \repo, a repository-centered workflow that packages data harmonization, a three-stage sEMG preprocessing stack, a twelve-feature window representation, shared model adapters, and benchmark utilities for multi-label finger-intent prediction on \physiomio~\citep{psrrepo2026,physiomio2026}. The current benchmark path aligns gesture labels with raw EMG, filters the signal with a 20--450 Hz zero-phase Butterworth stage, denoises it with wavelet soft-thresholding, segments 200 ms windows with 50\% overlap, and converts each window into a channel-by-feature tensor that can be consumed by LSTM, CNN, and graph neural network predictors. On the repository's held-out test artifacts, the LSTM achieves the strongest exact-match accuracy, finger accuracy, and macro AUROC (0.545, 0.784, and 0.858), while the GNN achieves the strongest macro F1 and AUPRC (0.706 and 0.776). The updated manuscript also records two open repository extensions: PR \#56 modularizes the 1D CNN training stack and latency-aware optimization, while PR \#59 adds teacher-ensemble distillation, threshold tuning, and local CPU latency benchmarking; because both are still under review, they are treated as pending extension evidence rather than merged headline results~\citep{psrpr56cnn2026,psrpr59distill2026}. These results position \repo\ as a useful systems benchmark rather than a single-model state-of-the-art claim, because the available evidence still covers one dataset and does not yet include full ablation or cross-dataset generalization studies. Even with that limitation, the released pipeline, metrics, pull-request notes, and paper-support artifacts establish a concrete base for future rehabilitation-oriented sEMG experiments.
